# Readme for Web Scraper App

## Description

This repository contains a web scraper app build with Ruby on Rails and the PostgreSQL DB

## Quickstart guide

$`git clone https://github.com/shirlaine/.git`

$`cd`

$`bundle install`

## Database Setup

- `$ bin/rake db:create`
